Kronika - API docs

Ogólne informacje

API służące do zapisu i odczytu wpisów do kroniki Parafii Matki Boskiej Częstochowskiej w Katowicach Podlesiu. Kronika znajduję się pod adresem http://fzskatowicepodlesie.pl/parafia/AAAA/.

Api znajduje się pod adresem adres.

GET /entries

Pobiera wszytkie wpisy do kroniki zapisane na serwerze.

Format zwracanych danych: json/text

Struktura zwracanego obiektu: [Entry, Entry, ..., Entry]

GET /entry/{id}

Pobiera wpis do kroniki o podanym id.

Param: id - id wpisu do kroniki np. 2.

Format zwracanych danych: json/text

Struktura zwracanego obiektu: Entry

GET /entries/year/{year}

Pobiera wszystkie wpisy do kroniki napisane w danym roku.

Param: year - rok w którym zostały napisane posty które chcemy pobrać np. 2019.

Format zwracanych danych: json/text

Struktura zwracanego obiektu: [Entry, Entry, ..., Entry]

GET /entry/years

Pobiera wszystkie lata w których były dodawane wpisy do kroniki.

Format zwracanych danych: json/text

Struktura zwracanego obiektu: tablica liczb [year1, year2, ..., yearN] np. [2005, 2007, 2021]
